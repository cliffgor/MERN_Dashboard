# Dashboard

### Technologies Used:
- Frontend: 
  - ReactJS
  - Axios
  - React Router Dom
  - Material UI
- Backend: 
  - NodeJS
  - Mongodb
  - ExpressJS
  - JWT
  - Passport JS

## Installation

This Project requires these technologies to run:

- [Node.js](https://nodejs.org/) v14+
- [MongoDB](https://mongodb.com/) you can use local or MongoDB Atlas

- after cloning the repository

```sh
cd MERN_auth
cd ../server 
npm i
npm run dev
cd client 
npm i
npm start
```
